比较不同的套餐

1. 词汇—产品特点

I—notes

对比不同产品的特点

1>. 服务常常提供不同的套餐。

Elektra is the basic package. (基础套餐)

Foto is our premium package. （高级套餐）

2>. 销售人员往往会从它们的优点来描述这些套餐。

The premium package is a high-speed service.

The premium package is high quality.

The basic package is a very good value for the money. （基础套餐非常划算）

I think the basic package will meet your needs.

II—选择正确的词

Elektra is the (money basic high) package.

Foto is our premium (needs package basic ).

The premium package is a high- (speed meet basic) service.

The premium package is high (basic service quality) .

The basic package is a very good (premium value need) for money.

I think the basic package will (meet speed value) your needs.

III—听音频、录音频

The basic package is very good value for the money.

The premium package is a high-speed service.

The premium package is high quality.

I think the basic package will meet your needs.

2. 语法—‘Both’和’neither’

I—阅读

L1 and L2 internet packages

From: benny.briggs@bigcompany.com

To: boss@bigcompany.com

Hi, boss.

I got you the information you needed about the internet packages:

  • Both L1 and L2 are high-speed packages.
  • L2 meets our needs for quality of performance more than L1.
  • Neither of them is really a great value for money. They’re both quite expensive.

Neither package is exactly what we want, to be honest. We should keep looking.

Best regards,

Benny

II—notes

Both 和 neither

1>. both 和 neither 可用于指代两个事物。both 用于肯定句。

Both L1 and L2 are high-speed packages.

2>. Neither 形成了两个事物之间的否定联系。

Neither package is really a great value for the money. (2个套餐都不是很划算)

3>. Similar、different 和 difference

以下词汇可用于比较所提供的服务。

A: Any other differences between the two packages?
B: The two packages are pretty similar, both high-speed.

A: Are the costs similar?
B: They’re not so different.

III—听音频、录音频

The packages are not so different. (套餐没有什么太大的不同)

The prices of the two services are similar.

The speeds are very different.

Neither is really a good value for the money. (这两者都不是物有所值。)

Both packages are equally high quality.

3. 表达—大的数字

I—notes

大的数字

1>. 我们通常不会在million，thousand和hundred结束时放s 。

It’s three million a year.

It’s four million yen per year.

2>. hundred 后面如果还有一个数字，可以使用 and 连接。(读的时候)

The price is 105,000 – a good value for money. (one hunfred and five thousand)

It’s 1,000,000 for the WFX, and 1,250,000 for the WF20. (one million, one million two hundred and fifty thousand )

3>. 您经常会在数字后面听到度量单位，例如 megabits per second (Mbps)。

The WFX is 15 megabits per second, and the WF20 is 20. (WFX 是 15 兆/秒，WF20 是 20 兆/秒)

II—听音频、录音频

The premium costs nine hundred and fifty thousand. (高级套餐九十五万。)

The basic package is five hundred thousand a year. (基础套餐一年五十万)

The total cost is one million, two hundred thousand. (总价是一百二十万)

That’s one hundred thousand per month. (每月十万。)
